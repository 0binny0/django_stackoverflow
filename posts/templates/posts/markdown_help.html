Inline links

To create an inline link within your post content, add a set of square brackets
which includes a title of the URL that you are linking to. Next, add a set of
parantheses to include the URL or which the link will direct to.

[link title](url) ====> link title

Paragraphs

Paragraphs are separated with a blank line both above and below
1 and more consecuative lines of text. To create a line break at the end of a line,
create two spaces and press the "return" key.

Inline styling


Italic: *hello world* or _hello world_
Bold: **hello world** or __hellow world__
Italic & Bold: ***hello world*** or __*hello world*__
